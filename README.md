# FloatDo

一款常驻桌面的悬浮球待办：随时可见，但尽量不打扰；想到就记，回车即收。

## 为什么是 FloatDo

很多时候你只需要一个“轻量提醒入口”，而不是再打开一个完整的知识库或项目管理工具。FloatDo 的定位是：把待办的捕获与查看成本降到最低，让你专注在正在做的事情上。

## 适合谁

- 开发者 / 高频电脑工作者
- 不想频繁切换窗口、但需要随手记任务的人
- 喜欢极简、常驻、可快速回收注意力的工作流

## 核心体验

- 桌面常驻一个小巧的悬浮球
- 点击展开待办面板
- 输入一句话，按 Enter 立即创建任务
- 勾选完成即可归档，保持列表清爽

## 功能亮点

- 悬浮球常驻与拖拽移动，支持边缘吸附
- 待办基础管理：新增 / 编辑 / 删除 / 完成
- 快速输入：单行输入 + 回车即创建
- 清单与筛选：按清单查看任务，区分未完成与已完成
- 本地数据：任务与清单保存到本地，不依赖云端账号

## 产品原则

- 低打扰：尽量不占用屏幕与注意力
- 高可见：需要时一键展开，随时可回看
- 高效率：捕获任务不超过 1 秒

## Roadmap（规划）

- 提醒与定时通知
- 开机自启动与托盘能力
- 更完善的数据备份与导入导出
- 更丰富的主题与个性化设置

## 技术栈（供参考）

- Python + PyQt（桌面端 UI）
- FastAPI（本地接口与数据层）
